## 🚀 Быстрый старт

```bash
# Добавить репозиторий
stplr repo add lumenara https://github.com/Cheviiot/Lumenara.git

# Обновить индекс
stplr refresh

# Установить пакет
stplr install <имя_пакета>
```

## 💻 Использование

### Установка пакета
```bash
stplr install happ
```

### Обновление всех пакетов
```bash
stplr upgrade
```

### Удаление пакета
```bash
stplr remove happ
```

### Поиск пакетов
```bash
stplr search git
```
